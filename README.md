💅 Nail Art 3D Booking Platform 

Minimalist tasarımın ve 3D etkileşimin kusursuz birleşimi. 

🚀 Öne Çıkan Özellikler 

3D İnteraktif Deneyim: Kullanıcıların randevu kartlarını 3D düzlemde hareket ettirebildiği, derinlik hissi veren modern arayüz. 

Akıcı 3D Silindirik Carousel: Hizmetlerin kavisli (cylindrical) bir perspektif içerisinde, drag & drop desteğiyle listelendiği gelişmiş navigasyon yapısı. 

Adımlı Rezervasyon (Wizard): Hizmet seçimi, tarih/saat yönetimi ve iletişim aşamalarından oluşan, hataya dayanıklı rezervasyon akışı. 

Minimalist Estetik: Zarif geçişler ve glow efektleriyle desteklenen, kullanıcıyı odak noktasında tutan koyu tema tasarımı.

Teknoloji

📦 Hızlı Başlangıç 

Projeyi yerel ortamınızda ayağa kaldırmak için terminalde şu komutları izleyin: 

Depoyu Klonlayın: 

git clone https://github.com/SoykanSOYLU/nail-art-3d-website.git 

Bağımlılıkları Yükleyin: 

npm install 

Projeyi Başlatın: 

npm run dev
